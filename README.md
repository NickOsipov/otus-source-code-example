# OTUS Source Code Example

Project Description

Setup Instructions
1. Clone the repository:
    ```bash
    git clone git@github.com:NickOsipov/otus-source-code-example.git
    cd otus-source-code-example
    ```
2. Create a project:
    ```bash
    uv python pin 3.11.11
    uv init
    uv venv
    uv add --requirements requirements.txt
    ```
